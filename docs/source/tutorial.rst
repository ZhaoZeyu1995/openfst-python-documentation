Installation
------------

** Not needed if you have followed the setup instructions for our ASR Labs **

.. role:: shell(code)
  :language: shell
  :class: highlight


To use openfst-python, we can simply install it with :shell:`pip`:

.. code-block:: console

   $ pip install openfst-python
